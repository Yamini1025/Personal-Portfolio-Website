# Personal Portfolio Website
A professional, responsive personal portfolio website built with HTML, CSS, and Bootstrap, featuring a Home page, About Me, Projects, and Contact sections. Includes a consistent header and footer, interactive CSS animations, 
and is currently being enhanced with JavaScript to make it more dynamic. This project was created as part of Udemy's 
[The Complete Full-Stack Web Development Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/).
